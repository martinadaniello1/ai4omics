## Requirements
The project is based on **Python** and **Jupyter Notebooks**.  
Make sure you have the following installed:  

- Python 3.8+  
- Jupyter Notebook / JupyterLab  
- Common scientific libraries:  
  - numpy  
  - pandas  
  - scikit-learn  
  - matplotlib  
  - seaborn  
  - scipy
